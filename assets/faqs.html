Why should I install WiFi Mouse?
WiFi Mouse is an remote control app for Android and IOS devices. It is designed to act as remote mouse, keyboard, trackpad, media player controller, internet Browser controller, Presentation controller and many other controller of PC/Mac.
Why should I install mouse server?
Please download mouse server from our official website: http://wifimouse.necta.us, then install it on your PC/Mac/Linux.
What are the features of Pro version?
WiFi Mouse Pro version features are media player controller, game controller, launcher application and full screen landscape simulate keyboard.
I’ve installed WiFi Mouse, how can I use it?
Please do the follow instructions：
1)：Install Mouse server on your PC/Mac/Ubuntu Linux from http://wifimouse.necta.us.
2): Run the Mouse server has installed.
3): Select one mouse server or input IP address of PC/Mac/Ubuntu Linux, then click connect button.
4): Now you can control your PC/Mac/Ubuntu Linux.
Which port does mouse server use?
TCP port 1978, please make sure firewall allow this port.
How do i use gestures?
At mouse/keyboard interface, there is touch area at the middle of screen. Now you can:
1): Move one finger for simulating mouse cursor movement.
2): Move two fingers up/down for simulating scroll up/down.
3): Tow fingers pinch in/out for simulating zoom in/out.
4): Tap one finger on screen for simulating mouse left button click.
5): Tap tow fingers on screen for simulating mouse right button click.
6): Press one finger and move for simulating drag and move.
7): Move three fingers also for simulating drag and move.
How do it use more controller feature?
After connected to your PC/Mac/Linux you can:
1): Media player controller: control media player running on PC/Mac/Ubuntu Linux.
2): Internet browser controller: control Internet Explorer, Chrome, Safari and other browsers, but some functions may not be enabled.
3): Presentation controller: control your PPT/Keynote for presentation.
4): Tasks controller: switch running applications windows.
5): Show remote applications and launcher any of them.
6): Shutdown controller: shutdown, sleep, restart, logout PC/Mac/Ubuntu Linux.
I have installed and run the mouse server, but WiFi Mouse show connect failed?
Please make sure the PC/Mac/Ubuntu Linux can connect to your phone, you can check the connection follow those instructions:
1): Get your phone's IP address from system setting -> Wi-Fi -> Tap the AP connected, it will show the IP address.
2): On you PC/Mac/Ubuntu Linux, run the command: ping [your phone's IP address].
3): If ping result is failed, please check firewall setting, allow mouse server and TCP port 1978.
4): If still connect to server failed, please change the AP settings for inner connection or other related settings.
